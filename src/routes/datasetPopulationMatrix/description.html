Description of datasetPopulationMatrix requests

GET /ngeo/datasetPopulationMatrix

From collections of collectionService, retrieve parameters and product an objet like :

            
            {
                datasetpopulationmatrix: {
                    criteriaTitles: ["keyword", "mission", "name", "sensor", "productType", "sensorMode"],
                    datasetPopulationValues: [
                        "",
                        "REMOTE",
                        collection.name, // name of collection
                        "REMOTE",
                        "REMOTE",
                        "REMOTE",
                        collection.id, // id of collection
                        collection.totalResults // count of products in collection
                    ]
                }
            }
